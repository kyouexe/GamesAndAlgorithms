Click cells to draw walls. Drag the S and E nodes to reposition them. Press Start to visualize.

Theory

Algorithm

A* Search uses a heuristic f(n) = g(n) + h(n) where g(n) is the actual cost from start and h(n) is an estimate to the goal.

State Space

Every non-wall cell in the 20×20 grid represents a state. The total state space is up to 400 nodes.

Initial State

The algorithm begins at the Start (S) node with g=0, h=Manhattan Distance.

Transition Model

From any cell, movement is possible to 4 adjacent non-wall cells (Up, Down, Left, Right) with a cost of 1.

Heuristic

Manhattan Distance: h(n) = |x₁-x₂| + |y₁-y₂|. Admissible for 4-directional grids — never overestimates.

Theory

How to Use

1. Click any region on the map to select it (it will glow).
2. Pick a valid color from the palette — conflicting colors are disabled.
3. The AI instantly solves the rest using CSP backtracking!

Algorithm

Constraint Satisfaction Problem (CSP) solved by Recursive Backtracking + Forward Checking to prune domains early.

State Space

A partial or complete assignment of colors to 6 variables (regions). With 4 colors, the raw space is 4⁶ = 4096 states.

Initial State

All regions are unassigned. Each has a domain of {Red, Green, Blue, Yellow}. Your pick fixes one variable.

Transition Model

A transition assigns a color from its domain to the next uncolored region. Forward Checking removes that color from adjacent domains.

Constraint

Binary constraint: for every edge (A, B) in the map graph, Color(A) ≠ Color(B).

Checkers — Minimax with Alpha-Beta Pruning

You play as ● Red. AI plays as ● Black (Dark). Click a piece to select, then click a highlighted square to move.

Red Pieces: 12 Black Pieces: 12 Turn: Red
Select a red piece to begin.

Theory

Algorithm

Minimax searches the game tree to find the move that maximizes AI's minimum guaranteed score. Alpha-Beta prunes branches that can't affect the final result.

State Space

Each board configuration (positions and types of all 24 pieces) is a unique state. The branching factor is ~7 typical moves.

Initial State

12 red pieces on rows 5-7 and 12 black pieces on rows 0-2 in a standard checkers setup.

Transition Model

Apply a valid move (slide or jump) to a board state to produce the successor state. Captured pieces are removed.

Evaluation Function

f(state) = (Black pieces) - (Red pieces). Kings count as 1.5. Goal states return ±∞.

Tower of Hanoi — Recursive State-Space Search

Choose the number of disks, then watch the optimal solution unfold step by step.

Move: 0 / 0

Theory

Algorithm

Recursive state-space search: solve(n, src, dst, aux). Move n-1 disks to aux, move disk n to dst, then move n-1 disks from aux to dst.

State Space

A state is the distribution of N disks across 3 pegs. Total states = 3ᴺ. For N=5, this is 243 unique configurations.

Initial State

All N disks are stacked on Peg A (Left) in descending order (largest at bottom).

Transition Model

Move the top disk from one peg to another, provided the destination peg is empty or its top disk is larger.

Optimal Moves

The minimum number of moves is 2ᴺ - 1. For 3 disks: 7, for 4: 15, for 5: 31 moves.
